# Kanala Geetha Pranathi | Portfolio

Welcome to the repository for my personal portfolio website! I am **Kanala Geetha Pranathi**, an Aspiring Full Stack AI Engineer passionate about developing AI-driven, data-powered solutions and applying modern technologies to solve real-world problems.

## 🚀 Overview

This portfolio showcases my journey, technical skills, projects, experience, and education. It serves as a centralized hub for anyone looking to understand my work, particularly in Full Stack Development and Artificial Intelligence.

## ✨ Features

- **Modern & Responsive Design:** Built with HTML5 and CSS3, ensuring a seamless experience across all devices (desktop, tablet, and mobile).
- **Hero Section:** A visually appealing introduction with quick links to projects and my GitHub profile.
- **About Me:** A professional summary of my goals, passions, and background.
- **Skills Section:** A categorized display of technical skills including Languages, Web Technologies, AI & Data, and Databases.
- **Projects Showcase:** Highlights of my major projects such as *NotebookLM Clone*, *ACE AI-Interview*, and *HouseHunt* with detailed tech stacks and GitHub links.
- **Experience & Education:** A structured timeline of my internships and academic history.
- **Certifications:** Badges of my continuous learning journey across various platforms.
- **Contact Form:** A fully functional contact form powered by [FormSubmit](https://formsubmit.co) for direct inquiries.

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, Google Fonts (Outfit, Inter)
- **Icons:** FontAwesome (v6.4.0)
- **Form Handling:** FormSubmit

## 💻 Getting Started

Since this is a static website, getting it up and running locally is extremely simple:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/GeethaPranathi/Portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd Portfolio
   ```

3. **Open the project:**
   Simply double-click on the `index.html` file to open it in your default web browser, or use a tool like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code for a better development experience.
